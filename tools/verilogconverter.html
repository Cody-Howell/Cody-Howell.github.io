Code Interpreter

Use parenthesis, and build up. This outputs a module.

(x0, x1..) Use numbers to represent variables (1 digit requirement)
( () ) ui interprets to the parenthesis
(&) k interprets to And
(|) j interprets to Or
(~, !) h interprets to Not
(^) l interprets to Xor
(?) z allows you to adjust it later.
