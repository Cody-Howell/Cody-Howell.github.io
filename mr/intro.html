This is the intro page for the Music Randomizer!

A quick explanation for what you see on the page below you:

  • Request (Red Box): The input box lets you set a string of nodes to play in a particular order without having to worry about the randomizer. Once you get going, the nodes that have been played will be listed beneath that box.
  • Graph View (Green Box): This box has the Begin and Stop buttons at the top, a selector to switch between different webs (read below), and the graph view of where you are and where you can go.
  • Next (Blue Box): This shows what is currently being played at the top, some of the Complex Web information (not explained in this document), what node is next, and where else you could have been.

There are 3 webs for you to experiment with:

  • Web 1: A linear web with a 5-10 minute playtime.
  • Web 2: A looping web with around 40 nodes and 5 starting points.
  • Web 3: A very small looping web with 8 nodes and 2 starting points.
